# Weighing Machine Problem

## Problem Statement
Mother, baby and dog standing on a weighing machine and it shows 170 pounds. How much does the baby weigh if the mother weighs 100 pounds more than the combined weight of the baby and the dog, and the dog weighs 60 percent less than the baby?

## Solution

Let's define the variables for their weights:
*   **$M$** = Mother's weight
*   **$B$** = Baby's weight
*   **$D$** = Dog's weight

From the problem description, we can set up the following equations:

1.  **"Mother, baby and dog standing on a weighing machine and it shows 170 pounds."**
    $$M + B + D = 170$$

2.  **"...the mother weighs 100 pounds more than the combined weight of the baby and the dog..."**
    $$M = 100 + (B + D)$$

3.  **"...and the dog weighs 60 percent less than the baby"**
    $$D = B - 0.60B = 0.40B$$

Now, let's solve the system of equations.

First, substitute Equation 2 into Equation 1:
$$(100 + B + D) + B + D = 170$$
$$100 + 2B + 2D = 170$$
$$2B + 2D = 70$$
$$B + D = 35$$

Next, substitute Equation 3 into our new simplified equation ($B + D = 35$):
$$B + 0.40B = 35$$
$$1.40B = 35$$
$$B = \frac{35}{1.40}$$
$$B = 25$$

**Conclusion:**
The baby weighs **25 pounds**.
